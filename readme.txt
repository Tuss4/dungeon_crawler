Description: Dungeon crawler!.

How to configure your map and rooms:

In simpleQuest.py in class Map, you can essentially change how your room
exits are set and your room names.

To play the game simply go to your terminal navigate to the parent directory
and type 'python simpleQuest.py'

To check out the test cases:
'python test_cases.py'

Enjoy!
